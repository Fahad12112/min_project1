Description

A simple application that displays tourist places with images and descriptions.

Used Widgets ✅

Required:

✅ ListView – to display a list of places
✅ AppBar – at the top of both screens
✅ Button – “Explore” and “Save Place” buttons
✅ Container – for cards and boxes
✅ SizedBox – for spacing between elements
✅ Image – tourist place images
✅ Text – text content
✅ MediaQuery – to adjust image sizes based on screen size
✅ Navigator – for navigation between screens

Screens
First Screen (HomeScreen):

AppBar displaying the app name

ListView containing 3 tourist places

Each place includes:

Image

Place name

Location

Short description

“Explore” button

Second Screen (DetailScreen):

AppBar displaying the place name

Large image

Detailed information about the place

“Save Place” button

Tourist Places:

Eiffel Tower – Paris

Great Wall of China – Beijing

Taj Mahal – India
